# 🏡 인테리어의 시작, 집다이브 🏡

###### 패스트캠퍼스에서 진행한 주어진 API를 분석하여 핀테크 서비스 만들기 프로젝트입니다.

**Javascript와 REST api를 활용한 인테리어 용품 쇼핑몰 집다이브**

**집다이브에 깊이 빠져보아요!**
